# <a name="choose-an-api-in-microsoft-graph-to-create-and-join-online-meetings"></a><span data-ttu-id="c4776-103">Escolher uma API no Microsoft Graph para criar e ingressar em reuniões online</span><span class="sxs-lookup"><span data-stu-id="c4776-103">Choose an API in Microsoft Graph to create and join online meetings</span></span>
