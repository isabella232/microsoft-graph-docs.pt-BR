# <a name="places-api-overview-preview"></a>Visão geral da API de locais (versão prévia)

É possível usar a API de locais para criar experiências de aplicativos atraentes com base em diferentes locais. Hoje, é possível ter acesso a locais da empresa e detalhes sobre cada um deles.

<!-- markdownlint-disable MD026 -->
## <a name="why-integrate-with-places"></a>Por que se integrar com lugares?
<!-- markdownlint-enable MD026 -->

Atualmente, os aplicativos modernos costumam ser aprimorados com reconhecimento de locais para ajudar a fornecer um contexto inteligente. Isso ajuda o usuário a navegar até um local ou permite que o aplicativo considere a localização de um usuário como parte da solução. A [API de locaisI](/graph/api/resources/place?view=graph-rest-beta) fornece informações detalhadas sobre locais, como salas ou listas de salas para habilitar esses cenários.

### <a name="help-customers-stay-synchronized-and-navigate-their-day"></a>Ajude os clientes a manterem-se sincronizados e a orientar seu dia de trabalho

Adicione locais ao calendário e ajude os clientes a orientar seu dia de trabalho e aumentar a produtividade.  Permita que os assistentes automatizados forneçam informações sobre as salas em um prédio. Envie mais detalhes para o seu convidado sobre como chegar a uma sala com um mapa via email. Crie aplicativos de setor de eventos que atenda e aproveite a [API de locais](/graph/api/resources/place?view=graph-rest-beta) A API que fornece a configuração de sala.
