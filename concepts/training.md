# <a name="microsoft-graph-training"></a>Treinamento do Microsoft Graph

Exercícios de treinamento do Microsoft Graph são tutoriais passo a passo que orientá-lo a criar um aplicativo básico que usa a autenticação do Windows Azure Active Directory e acessa os dados por meio do Microsoft Graph.

Os seguintes tutoriais de treinamento estão disponíveis no momento.

- [ASP.NET](https://docs.microsoft.com/graph/training/aspnet-tutorial/)
- [Microsoft Flow](https://docs.microsoft.com/graph/training/flow-tutorial/)
- [Node.js](https://docs.microsoft.com/graph/training/node-tutorial/)
- [PHP](https://docs.microsoft.com/graph/training/php-tutorial/)
- [Python](https://docs.microsoft.com/graph/training/python-tutorial/)
- [Ruby](https://docs.microsoft.com/graph/training/ruby-tutorial/)
- [React](https://docs.microsoft.com/graph/training/react-tutorial/)
- [UWP](https://docs.microsoft.com/graph/training/uwp-tutorial/)

Não vê seu idioma preferencial ou outra plataforma? Solicite um tutorial para ele sobre [UserVoice](https://officespdev.uservoice.com/forums/224641-feature-requests-and-feedback/category/101632-microsoft-graph-o365-rest-apis).
