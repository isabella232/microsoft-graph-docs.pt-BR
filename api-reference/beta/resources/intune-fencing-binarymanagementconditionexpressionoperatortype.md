Operadores binários suportados para expressões de condição de gerenciamento.

## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|ou|,0|Avalia um conjunto de operandos como true se e somente se um ou mais de seus operandos for true.|
|and|1|Avalia um conjunto de operandos como true se e somente se todos os seus operandos forem verdadeiros.|
