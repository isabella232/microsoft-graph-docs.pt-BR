# <a name="chartseriesformat-resource-type"></a>Tipo de recurso ChartSeriesFormat

Abrange as propriedades de formatação da série do gráfico.


## <a name="methods"></a>Métodos
Nenhum

## <a name="properties"></a>Propriedades
Nenhuma

## <a name="relationships"></a>Relações
| Relação | Tipo	    |Descrição|
|:---------------|:--------|:----------|
|fill|[ChartFill](chartfill.md)|Representa o formato de preenchimento de uma série do gráfico, que inclui informações sobre a formatação da tela de fundo. Somente leitura.|
|line|[ChartLineFormat](chartlineformat.md)|Representa a formatação de linha. Somente leitura.|
