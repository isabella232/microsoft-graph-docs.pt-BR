# <a name="devicemanagementtroubleshootingerrordetails-resource-type"></a>tipo de recurso de deviceManagementTroubleshootingErrorDetails

> **Importante:** APIs sob a versão /beta no Microsoft Graph estão sujeitos a alterações. Não há suporte para o uso dessas APIs em aplicativos de produção.

> **Observação:** A API do Microsoft Graph para Intune requer uma [licença de Intune ativa](https://go.microsoft.com/fwlink/?linkid=839381) para o inquilino.

Objeto que contém informações detalhadas sobre o erro e suas atualizações.

## <a name="properties"></a>Propriedades
|Propriedade|Tipo|Descrição|
|:---|:---|:---|
|context|String|Ainda não documentado|
|Falha|String|Ainda não documentado|
|failureDetails|String|A descrição detalhada do que deu errada.|
|remediação|String|A descrição detalhada de como corrigir esse problema.|
|recursos|coleção [deviceManagementTroubleshootingErrorResource](../resources/intune-troubleshooting-devicemanagementtroubleshootingerrorresource.md)|Links para documentação úteis sobre essa falha.|

## <a name="relationships"></a>Relações
Nenhum
