# <a name="macosgatekeeperappsources-enum-type"></a>tipo de enum macOSGatekeeperAppSources

> **Importante:** APIs sob a versão /beta no Microsoft Graph estão sujeitos a alterações. Não há suporte para o uso dessas APIs em aplicativos de produção.

> **Observação:** A API do Microsoft Graph para Intune requer uma [licença de Intune ativa](https://go.microsoft.com/fwlink/?linkid=839381) para o inquilino.

Opções de fonte do aplicativo para macOS Gatekeeper.

## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|não-configuradas|0|Valor de padrão de dispositivo, sem intenção.|
|macAppStore|1|Somente aplicativos a partir do AppStore Mac podem ser executados.|
|macAppStoreAndIdentifiedDevelopers|2|Podem ser executados somente aplicativos do Mac AppStore e desenvolvedores identificados.|
|em qualquer lugar|3|Aplicativos de qualquer lugar podem ser executados.|
