# <a name="androiddeviceownerappautoupdatepolicytype-enum-type"></a>tipo de enum androidDeviceOwnerAppAutoUpdatePolicyType

> **Importante:** APIs sob a versão /beta no Microsoft Graph estão sujeitos a alterações. Não há suporte para o uso dessas APIs em aplicativos de produção.

> **Observação:** A API do Microsoft Graph para Intune requer uma [licença de Intune ativa](https://go.microsoft.com/fwlink/?linkid=839381) para o inquilino.

Os valores possíveis proprietário do dispositivo Android para estados da automático do aplicativo do dispositivo atualizam a política.

## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|não-configuradas|0|Não configurado; Este valor será ignorado.|
|userChoice|1|O usuário pode controlar atualizações automáticas.|
|nunca|2|Aplicativos nunca são atualizados para automático.|
|wiFiOnly|3|Aplicativos atualizado automaticamente por Wi-Fi só estarão.|
|sempre|4|Aplicativos são atualizado automaticamente a qualquer momento. Tarifas de dados podem ser aplicadas.|
