> **Observação:** A API do Microsoft Graph para Intune requer uma [licença de Intune ativa](https://go.microsoft.com/fwlink/?linkid=839381) para o inquilino.

Atualizar tipos de sistema para o proprietário de dispositivo Android.

## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|deviceDefault|0|Comportamento de padrão de dispositivo, que geralmente solicita que o usuário aceite atualizações de sistema.|
|Adiar|1|Adie a instalação automática das atualizações de backup para 30 dias.|
|em janelas|2|Instale automaticamente dentro de uma janela de manutenção diária.|
|automatic|3|Instale automaticamente atualizações assim que possível.|
