# <a name="prereleasefeatures-enum-type"></a>tipo de enum prereleaseFeatures

> **Importante:** APIs sob a versão /beta no Microsoft Graph estão sujeitos a alterações. Não há suporte para o uso dessas APIs em aplicativos de produção.

> **Observação:** A API do Microsoft Graph para Intune requer uma [licença de Intune ativa](https://go.microsoft.com/fwlink/?linkid=839381) para o inquilino.

Valores possíveis para recursos de pré-lançamento.

## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|userDefined|0|Definido pelo usuário, valor padrão, sem intenção.|
|settingsOnly|1|Recursos de apenas pré-lançamento configurações.|
|settingsAndExperimentations|2|Recursos de pré-lançamento experimentations e configurações.|
|notAllowed|3|Recursos de pré-lançamento não permitidos.|
