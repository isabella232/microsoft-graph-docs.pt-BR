# <a name="educationassignmentclassrecipient-resource-type"></a>tipo de recurso de educationAssignmentClassRecipient

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Usada dentro da propriedade [assignment.assignTo](educationassignment.md) . Quando definido como destinatário de classe, cada aluno na classe receberá um objeto de envio quando a atribuição é publicada.

Este recurso é uma subclasse de [educationAssignmentRecipient](educationassignmentrecipient.md).

## <a name="properties"></a>Propriedades

Nenhum.
