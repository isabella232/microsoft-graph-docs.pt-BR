# <a name="accessreviewinstancedecisionitemtarget-resource-type"></a>tipo de recurso accessReviewInstanceDecisionItemTarget

Namespace: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Representa uma classe base para diferentes tipos de destinos de revisão, cada um com suas próprias propriedades específicas. Os tipos de destino suportados são: [accessReviewInstanceDecisionItemUserTarget](accessreviewinstancedecisionitemusertarget.md) e [accessReviewInstanceDecisionItemServicePrincipalTarget](accessreviewinstancedecisionitemserviceprincipaltarget.md).

## <a name="properties"></a>Propriedades
Nenhum.
