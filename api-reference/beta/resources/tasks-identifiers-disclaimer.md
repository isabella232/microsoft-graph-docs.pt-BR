[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

<span data-ttu-id="3ec5d-108">Identificadores de objetos em tarefas são valores de cadeia de caracteres gerados pelo serviço.</span><span class="sxs-lookup"><span data-stu-id="3ec5d-108">Identifiers for objects in Tasks are service-generated string values.</span></span> <span data-ttu-id="3ec5d-109">.</span><span class="sxs-lookup"><span data-stu-id="3ec5d-109"></span></span> <span data-ttu-id="3ec5d-110">Os valores são 28 caracteres de comprimento e diferenciam maiusculas de minúsculas.</span><span class="sxs-lookup"><span data-stu-id="3ec5d-110">The values are 28 characters long and are case-sensitive.</span></span> <span data-ttu-id="3ec5d-111">Quando passados como no, o serviço fará uma validação de formato simples do identificador, se o formato validação falhar, os chamadores receberá uma resposta de erro de solicitação incorreta (400) indicando esse problema.</span><span class="sxs-lookup"><span data-stu-id="3ec5d-111">When passed as in, the service will do a simple format validation of the identifier, if the format validation fails, the callers will receive a Bad Request (400) error response indicating this problem.</span></span> <span data-ttu-id="3ec5d-112">Receber esse erro indica um bug do aplicativo de chamada, tais como:</span><span class="sxs-lookup"><span data-stu-id="3ec5d-112">Receiving this error indicates a bug in the calling application, such as:</span></span>

- <span data-ttu-id="3ec5d-p103">O aplicativo de chamada processou o identificador como uma cadeia de caracteres que não diferencia maiúsculas de minúsculas. Os identificadores em Tarefas diferenciam maiúsculas de minúsculas.</span><span class="sxs-lookup"><span data-stu-id="3ec5d-p103">The calling application processed the identifier as a case insensitive string. Identifiers in Tasks are case-sensitive.</span></span>
- <span data-ttu-id="3ec5d-p104">O aplicativo de chamada truncou o identificador. Os identificadores em Tarefas têm 28 caracteres.</span><span class="sxs-lookup"><span data-stu-id="3ec5d-p104">The calling application truncated the identifier. Identifiers in Tasks are 28 characters long.</span></span>
- <span data-ttu-id="3ec5d-p105">O aplicativo de chamada tentou gerar um valor do identificador para um objeto em Tarefas. Não são aceitos identificadores gerados por clientes. Todos os identificadores são gerados pelo serviço durante a criação de objetos.</span><span class="sxs-lookup"><span data-stu-id="3ec5d-p105">The calling application attempted to generate an identifier value for an object in Tasks. <span data-ttu-id="3ec5d-p106">A validação **não é um recurso de segurança**.
