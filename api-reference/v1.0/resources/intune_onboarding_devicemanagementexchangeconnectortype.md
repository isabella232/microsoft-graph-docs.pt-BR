# <a name="devicemanagementexchangeconnectortype-enum-type"></a>tipo de enum deviceManagementExchangeConnectorType

> **Observação:** O uso das APIs do Microsoft Graph para configurar controles e políticas do Intune ainda exige que o serviço do Intune seja [corretamente licenciado](https://go.microsoft.com/fwlink/?linkid=839381) pelo cliente.

O tipo de conector do Exchange.
## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|onPremises|0|Conecta-se ao ambiente do Exchange local.|
|hospedado|1|Conecta-se ao ambiente do Exchange de multilocatário O365|
|serviceToService|2|Intune Service se conectará diretamente ao ambiente do Exchange de multilocatário O365|
|dedicada|3|Conecta-se ao ambiente do Exchange dedicada do O365.|
