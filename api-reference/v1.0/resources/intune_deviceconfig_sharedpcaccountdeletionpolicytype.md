# <a name="sharedpcaccountdeletionpolicytype-enum-type"></a>tipo enumerado sharedPCAccountDeletionPolicyType

> **Observação:** O uso das APIs do Microsoft Graph para configurar controles e políticas do Intune ainda exige que o serviço do Intune seja [corretamente licenciado](https://go.microsoft.com/fwlink/?linkid=839381) pelo cliente.

Valores possíveis para quando contas são excluídas em um PC compartilhado.
## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|imediato|0|Excluir imediatamente.|
|diskSpaceThreshold|1|Excluir no limite de espaço em disco.|
|diskSpaceThresholdOrInactiveThreshold|2|Excluir no limite de espaço em disco ou no limite inativo.|
