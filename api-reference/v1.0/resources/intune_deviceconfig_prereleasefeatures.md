# <a name="prereleasefeatures-enum-type"></a>Tipo enumerado prereleaseFeatures

> **Observação:** O uso das APIs do Microsoft Graph para configurar controles e políticas do Intune ainda exige que o serviço do Intune seja [corretamente licenciado](https://go.microsoft.com/fwlink/?linkid=839381) pelo cliente.

Valores possíveis para recursos de pré-lançamento.
## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|userDefined|0|Definido pelo usuário, valor padrão, sem intenção.|
|settingsOnly|1|Recursos de pré-lançamento somente para configurações.|
|settingsAndExperimentations|2|Recursos de pré-lançamento para experimentações e configurações.|
|notAllowed|3|Recursos de pré-lançamento não permitidos.|
