# <a name="driveitem-resource-type"></a><span data-ttu-id="425a8-104">tipo de recurso driveItem</span><span class="sxs-lookup"><span data-stu-id="425a8-104">driveItem resource type</span></span>

<span data-ttu-id="425a8-p102">O recurso **DriveItem** representa um arquivo, pasta ou outro item armazenado em uma unidade. * <span data-ttu-id="425a8-113">As pastas têm uma [**faceta folder**][folder]</span><span class="sxs-lookup"><span data-stu-id="425a8-113">Folders have a [**folder facet**][folder]</span></span>
