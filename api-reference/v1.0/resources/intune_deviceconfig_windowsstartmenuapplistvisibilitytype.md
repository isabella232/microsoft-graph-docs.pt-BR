# <a name="windowsstartmenuapplistvisibilitytype-enum-type"></a>tipo enumerado windowsStartMenuAppListVisibilityType

Tipo de visibilidade de lista de aplicativos do menu Iniciar.
## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|userDefined|0|Definido pelo usuário. Valor padrão.|
|collapse|1|Recolhe a lista de aplicativos no menu Iniciar.|
|remove|2|Remove inteiramente a lista de aplicativos do menu Iniciar.|
|disableSettingsApp|4|Desabilita a alternância correspondente (recolher ou remover) na configuração do aplicativo.|
