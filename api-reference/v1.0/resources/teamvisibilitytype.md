#<a name="teamvisibilitytype-enum-type"></a>tipo de enum teamVisibilityType



Descreve a visibilidade de uma [equipe](../resources/team.md). 

## <a name="members"></a>Membros

| Membro | Valor| Descrição |
|:---------------|:--------|:----------|
|privada|0|Qualquer pessoa pode ver a equipe, mas apenas o proprietário pode adicionar um usuário para a equipe.|
|public|1|Qualquer pessoa pode participar da equipe.|
