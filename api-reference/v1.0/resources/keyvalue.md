# <a name="keyvalue-resource-type"></a>tipo de recurso KeyValue

Namespace: microsoft.graph

| Propriedade     | Tipo   |Descrição|
|:---------------|:--------|:----------|
|key|string| Chave para o par chave-valor. |
|value|string| Valor para o par chave-valor.|
