Rótulos de classificação de conteúdo de TV na Irlanda
## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|allAllowed|0|Valor padrão, permitir todo o conteúdo dos programas de TV|
|allBlocked|1|Não permitir conteúdo algum de programas de TV|
|geral|2|A classificação GA é adequada para todos os públicos|
|crianças|3|A classificação CH é adequada para crianças|
|youngAdults|4|A classificação YA é adequada para o público adolescente|
|parentalSupervision|5|A classificação PS convida pais e responsáveis a considerar a restrição de acesso às crianças|
|adulto|6|A classificação MA é adequada para adultos|
