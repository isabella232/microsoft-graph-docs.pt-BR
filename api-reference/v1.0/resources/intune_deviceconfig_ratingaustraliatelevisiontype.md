Rótulos de classificação de conteúdo de TV na Austrália
## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|allAllowed|0|Valor padrão, permitir todo o conteúdo dos programas de TV|
|allBlocked|1|Não permitir conteúdo algum de programas de TV|
|preschoolers|2|A classificação P destina-se a pré-escolares|
|children|3|A classificação C destina-se a crianças com menos de 14 anos|
|general|4|A classificação G é adequada para todas as idades|
|parentalGuidance|5|A classificação PG é recomendada para telespectadores jovens|
|mature|6|A classificação M é recomendada para telespectadores com mais de 15 anos|
|agesAbove15|7|A classificação MA15+ não é adequada para telespectadores com menos de 15 anos|
|agesAbove15AdultViolence|8|A classificação AV15+ não é adequada para telespectadores com menos de 15 anos, violência específica para adultos|
