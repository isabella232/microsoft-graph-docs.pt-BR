# <a name="rgbcolor-resource-type"></a>Tipo de recurso rgbColor

Cor em RGB.
## <a name="properties"></a>Propriedades
|Propriedade|Tipo|Descrição|
|:---|:---|:---|
|r|Byte|Valor de vermelho|
|g|Byte|Valor de verde|
