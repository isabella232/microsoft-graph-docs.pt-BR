# <a name="vpptokensyncstatus-enum-type"></a>tipo enumerado vppTokenSyncStatus

> **Observação:** O uso das APIs do Microsoft Graph para configurar controles e políticas do Intune ainda exige que o serviço do Intune seja [corretamente licenciado](https://go.microsoft.com/fwlink/?linkid=839381) pelo cliente.

Status de sincronização possíveis associados a um token de programa de compra de Volume do Apple.
## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|nenhum|0|Status padrão.|
|inProgress|1|Última sincronização em andamento.|
|concluída|2|Última sincronização concluída com êxito.|
|failed|3|Última sincronização falhou.|
