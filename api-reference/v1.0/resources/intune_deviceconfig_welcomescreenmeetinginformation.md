# <a name="welcomescreenmeetinginformation-enum-type"></a>tipo de enum welcomeScreenMeetingInformation

> **Observação:** O uso das APIs do Microsoft Graph para configurar controles e políticas do Intune ainda exige que o serviço do Intune seja [corretamente licenciado](https://go.microsoft.com/fwlink/?linkid=839381) pelo cliente.

Valores possíveis para informações da reunião de tela de boas-vindas.
## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|userDefined|0|Definido pelo usuário, valor padrão, sem intenção.|
|showOrganizerAndTimeOnly|1|Mostra organizador e horário apenas.|
|showOrganizerAndTimeAndSubject|2|Mostrar o organizador, hora e assunto (assunto é oculto para as reuniões privadas).|
