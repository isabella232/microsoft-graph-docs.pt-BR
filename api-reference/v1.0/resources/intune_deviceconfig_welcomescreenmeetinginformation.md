# <a name="welcomescreenmeetinginformation-enum-type"></a>Tipo enumerado welcomeScreenMeetingInformation

> **Observação:** O uso das APIs do Microsoft Graph para configurar controles e políticas do Intune ainda exige que o serviço do Intune seja [corretamente licenciado](https://go.microsoft.com/fwlink/?linkid=839381) pelo cliente.

Valores possíveis para as informações da reunião na tela de boas-vindas.
## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|userDefined|0|Definido pelo usuário, valor padrão, sem intenção.|
|showOrganizerAndTimeOnly|1|Mostrar somente o horário e o organizador.|
|showOrganizerAndTimeAndSubject|2|Mostrar o organizador, o horário e o assunto (o assunto é ocultado para as reuniões privadas).|
