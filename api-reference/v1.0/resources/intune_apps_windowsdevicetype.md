# <a name="windowsdevicetype-enum-type"></a>Tipo enumerado windowsDeviceType

> **Observação:** O uso das APIs do Microsoft Graph para configurar controles e políticas do Intune ainda exige que o serviço do Intune seja [corretamente licenciado](https://go.microsoft.com/fwlink/?linkid=839381) pelo cliente.

Contém propriedades para o tipo de dispositivo Windows.
## <a name="windowsdevicetype-flags"></a>sinalizadores de windowsDeviceType
|Membro|Valor|Descrição|
|:---|:---|:---|
|none|0|Nenhum sinalizador definido.|
|desktop|1|Se o tipo de dispositivo área de trabalho do Windows é suportado ou não.|
|mobile|2|Se o tipo de dispositivo Windows Mobile é suportado ou não.|
|holographic|4|Se o tipo de dispositivo Windows Holographic é suportado ou não.|
|team|8|Se o tipo de dispositivo Windows Teams é suportado ou não.|
