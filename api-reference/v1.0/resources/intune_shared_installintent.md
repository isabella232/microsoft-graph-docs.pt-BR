# <a name="installintent-enum-type"></a>tipo enumerado installIntent

> **Observação:** O uso das APIs do Microsoft Graph para configurar controles e políticas do Intune ainda exige que o serviço do Intune seja [corretamente licenciado](https://go.microsoft.com/fwlink/?linkid=839381) pelo cliente.

Valores possíveis para a intenção de instalar escolhida pelo administrador.
## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|available|0|Intenção de instalação disponível.|
|required|1|Intenção de instalação obrigatória.|
|uninstall|2|Desinstalar a intenção de instalação.|
|availableWithoutEnrollment|3|Disponível sem inscrição de intenção de instalação.|
