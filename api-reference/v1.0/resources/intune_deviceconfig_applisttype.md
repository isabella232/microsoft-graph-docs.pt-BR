# <a name="applisttype-enum-type"></a>tipo enumerado appListType

> **Observação:** O uso das APIs do Microsoft Graph para configurar controles e políticas do Intune ainda exige que o serviço do Intune seja [corretamente licenciado](https://go.microsoft.com/fwlink/?linkid=839381) pelo cliente.

Possíveis valores da lista de aplicativos de conformidade.
## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|nenhum|0|Valor padrão, sem intenção.|
|appsInListCompliant|1|A lista representa os aplicativos que serão considerados compatíveis (somente aplicativos na lista são compatíveis).|
|appsNotInListCompliant|2|A lista representa os aplicativos que serão considerados não compatíveis (todos os aplicativos são compatíveis, exceto os aplicativos na lista).|
